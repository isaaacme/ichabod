How it works איך זה עובד

Up in four steps. ארבעה שלבים בלבד.

No manual installs, no config files to write, no ports to configure. ללא התקנות ידניות, ללא קבצי הגדרות לכתיבה, ללא הגדרת פורטים.

1

Run npx ichabod initהרץ npx ichabod init

An interactive wizard asks for project name, framework, site language, and admin credentials.אשף אינטראקטיבי שואל שם פרויקט, framework, שפת אתר ופרטי מנהל.

2

PocketBase is downloaded & startedPocketBase מורד ומופעל

The correct binary for your OS is fetched from GitHub releases, extracted, and launched — no manual install.הקובץ המתאים למערכת ההפעלה שלך מורד מ-GitHub Releases, מחולץ ומופעל — ללא התקנה ידנית.

3

Schema & content created automaticallyסכמה ותוכן נוצרים אוטומטית

site_settings and posts collections are created with full SEO fields. Two starter posts are seeded.אוספי site_settings ו-posts נוצרים עם שדות SEO מלאים. שני פוסטי התחלה נזרעים.

4

Everything stays runningהכל נשאר פעיל

PocketBase and your framework dev server stay alive after init. Use npx ichabod start next time.PocketBase ושרת הפיתוח של ה-framework נשארים פעילים לאחר ההקמה. בפעם הבאה השתמש ב-npx ichabod start.
